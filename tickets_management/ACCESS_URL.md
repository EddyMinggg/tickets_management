# 🌍 立即訪問應用 - 3 種方法

## ⚡ 最快方法：本地網路

### 步驟 1：獲取電腦 IP

Windows 命令行執行：
```cmd
ipconfig
```

找到類似 `192.168.1.XXX` 的 IPv4 地址

### 步驟 2：手機連接同一 WiFi

### 步驟 3：訪問應用

在手機瀏覽器輸入：
```
http://192.168.1.XXX:8000
```

替換 `192.168.1.XXX` 為您的電腦 IP

✅ 完成！現在可以安裝到主螢幕

---

## 🔗 方法 2：Ngrok（公開網址）

### 一鍵設置

1. **下載 Ngrok**
   https://ngrok.com/download

2. **解壓並運行**
   ```cmd
   ngrok http 8000
   ```

3. **複製網址**
   ```
   Forwarding  http://abc123def.ngrok.io -> http://localhost:8000
   ```

4. **分享網址**
   可以給任何人訪問！

✅ 獲得公開網址，可分享給全球用戶
